# 🧠 LLM Skills Vault

A collection of high-performance system prompts and skill definitions designed to extend the capabilities of AI models. These allow your LLM to act as a specialized agent with specific guardrails and workflows.

## 🛠️ Usage
1. **Browse** the available skills in this repo.
2. **Copy** the desired definition block.
3. **Paste** it into your AI’s system instructions or project context files (e.g., `.clauderc`, `.cursorrules`).
